Quiz Game Description

This is a web-based quiz game built with HTML, CSS, and JavaScript, featuring:
A multiple-choice question system with shuffled questions from HTML, CSS, and JavaScript.
A modal popup to enter the player's name before starting the game.
A real-time game timer and background music. A progress bar that updates with each question answered.
Audio feedback for correct answers.
Tracks correct and incorrect attempts, and shows a final score.
